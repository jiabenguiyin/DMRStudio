## 产品定位

`DMR Studio` 是一款闭源桌面产品。

它的目标不是让用户研究复杂配置，而是尽量把录制、渲染、上传、任务管理和常用设置整理成更容易上手的桌面流程，让新手也能更顺畅地开始使用。

我们更重视这些体验：

- 新手第一次接触时也能尽快看懂界面
- 常见操作尽量通过按钮和表单完成
- 安装、启动、使用流程尽量接近日常 Windows 软件
- 后续版本持续增加功能，并长期维护更新

## 交付方式


标准使用流程会尽量保持为常见软件的体验：

1. 下载安装包
2. 运行安装程序
3. 选择安装路径，或直接使用默认路径
4. 安装完成后自动创建桌面快捷方式
5. 双击桌面图标启动程序
6. 进入桌面应用后正常使用

这也是当前产品封装阶段要达成的核心目标之一。

## 产品特点

- 面向新手：尽量减少手改配置和目录的负担
- 桌面化体验：以桌面窗口、安装器、快捷方式为主要交付形式
- 更容易操作：把常见功能放进统一界面
- 持续维护：后续版本会继续补充和完善更多功能
- 版本可更新：后续将结合 GitHub Releases 做版本发布与更新提示

## 当前方向

当前版本正在继续收口桌面产品体验，重点包括：

- 统一任务管理入口
- 优化运行控制与状态查看
- 完善安装包与版本发布流程
- 提升首次安装后的直接可用性
- 为后续新功能迭代预留稳定基础

## 安装与启动体验

`DMR Studio` 预期采用标准 Windows 安装流程：

- 支持默认安装目录
- 支持用户手动选择安装目录
- 安装后创建桌面快捷方式
- 双击快捷方式即可启动
- 启动后直接进入桌面应用主界面

整体目标就是让用户像安装和使用普通 Windows 软件一样使用它。
